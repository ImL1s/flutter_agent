BuiltInActions

BuiltInActions class

Registers standard built-in actions that map to Flutter SemanticsActions.

Call BuiltInActions.registerDefaults(registry) to populate an ActionRegistry with the common actions: tap, longPress, scrollUp, scrollDown, scrollLeft, scrollRight, setText, focus, dismiss.

final registry = ActionRegistry();
BuiltInActions.registerDefaults(registry);
